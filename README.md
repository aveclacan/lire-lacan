# Lire Lacan

Leitor bilíngue dos Seminários de Jacques Lacan — francês original com tradução para o português, notas de homofonia e ferramentas de estudo.

**Acesso:** [seuusuario.github.io/lire-lacan](https://seuusuario.github.io/lire-lacan)

---

## Funcionalidades

- Leitura contínua e bilíngue (FR → PT) dos Seminários
- Notas de homofonia e jogos de linguagem lacanianos
- Áudio com pronúncia em francês parisiense
- Marcações, destaques e comentários no texto
- Salvamento de trechos de interesse
- Progresso de leitura em porcentagem

## Seminários disponíveis

Coloque os arquivos JSON gerados pelo processador na pasta `seminarios/`:

```
seminarios/
  seminario_01.json
  seminario_07.json
  seminario_11.json
  seminario_20.json
  seminario_23.json
  ...
```

## Estrutura

```
lire-lacan/
  index.html          ← leitor bilíngue (abrir no navegador)
  seminarios/         ← JSONs dos seminários processados
  assets/             ← recursos futuros
  README.md
```
